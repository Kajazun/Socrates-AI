# Socrates: AI-Powered Learning Engine

Socrates is an advanced educational tool built on the Socratic Method. It transforms passive reading into an active, critical dialogue, ensuring deep understanding of PDF materials.

### Key Features

* **Socratic Dialogue:** The AI challenges you with probing questions instead of providing direct answers.
* **Truth-Check Logic:** Real-time detection of factual and logical contradictions in user responses.
* **Dynamic Knowledge Mapping:** Visualizes the learning journey using Mermaid.js.
* **Performance Analysis:** Provides an objective summary of strengths and weaknesses.
* **Flashcard Generation:** Automatically creates study cards based on the conversation history.

### Tech Stack

* **Language:** Python
* **LLM:** Llama 3.3-70b (via Groq API)
* **UI Framework:** Gradio
* **Visualization:** Mermaid.js

### Installation & Usage

1. Clone the repository.
2. Install dependencies:
```bash
pip install -r requirements.txt

```


3. Set up your API key:
```bash
export GROQ_API_KEY='your_api_key'

```


4. Run the application:
```bash
python app.py

```
